# CKEditor CRDT Collaboration

This is a simple experiment to add [CRDT](https://en.wikipedia.org/wiki/Conflict-free_replicated_data_type) based collaboration to the ck-editor.

Don't use this in production!

## Development

Run

```bash
yarn ws-server
```

```bash
yarn dev
```

To get started
